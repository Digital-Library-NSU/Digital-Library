1. Перейдите в папку фронтенда
2. Если у вас не установлены node, npm и angular, то установите.
3. Введите в папке фронтенда

```
npm install
```

чтобы скачать зависимости

4. Для запуска сайта команда

```
ng serve -o
```

-o сразу откроет в браузере

5. Путь к бекенду захардкожен в src/environments/environment.ts как http://localhost:8000, если у вас другой порт, то поменяйте

6. Если какой-то запрос упадёт с ошибкой из-за cors политики, то меняйте ваш бекенд, чтобы принимал разные источники
